## Build (FFmpeg configure — no GPL components)

```sh
./configure \
  --enable-videotoolbox \
  --enable-libvpx --enable-libopus --enable-libvorbis --enable-libdav1d \
  --enable-static --disable-shared \
  --disable-doc --disable-ffplay --disable-debug \
  --disable-libxcb --disable-libxcb-shm --disable-libxcb-xfixes \
  --disable-libxcb-shape --disable-xlib
make -j$(sysctl -n hw.ncpu)
# built per arch against universal2 deps, then lipo'd into a universal binary
```

> No `--enable-gpl`, no `--enable-version3`, no x264/x265 — FFmpeg remains LGPLv2.1.
> Generated by the `build-ffmpeg.yml` workflow; `<SHA256 …>` is filled in automatically.
